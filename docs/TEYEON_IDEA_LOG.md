# TEYEON Idea Log

앱 개발 중 떠오르는 아이디어를 보관하는 문서입니다. 확정된 작업은 `TEYEON_ROADMAP.md`로 옮겨 우선순위를 관리합니다.

## 디자인

- Luxury Sports Club Dashboard 방향으로 전체 톤 개선
- 경기 운영 화면은 현장 사용성을 우선하고, 정보 밀도와 가독성을 높이기
- 관리자/운영자 화면은 조용하고 단단한 컨트롤 센터 느낌으로 정리
- 멤버와 게스트가 보는 화면은 프리미엄 클럽 앱처럼 명확하고 가볍게 구성

## KDK

- KDK Realtime 동기화 성공 상태 유지
- 경기 투입, 점수 저장, 다른 브라우저 자동 반영 흐름 안정화
- MEMBER/GUEST도 KDK 생성 과정을 볼 수 있는 Preview Mode 장기 과제로 검토
- Preview Mode에서는 실제 저장 없이 대진 생성 흐름과 결과를 체험하게 하기
- 운영자는 실시간 경기 상태와 랭킹을 빠르게 확인할 수 있게 개선

## 로그인/권한

- 카카오 로그인은 유지
- 로그인 후 현재 경로로 돌아오도록 redirect 유지
- 앱 권한 판단 기준은 `profiles.role`로 단일화
- 신규 로그인자는 기본 `GUEST`
- CEO가 관리자 화면에서 `MEMBER`/`ADMIN` 권한을 수동 부여
- 카카오 닉네임, 이메일, 전화번호 자동 매칭은 권한 판단에 사용하지 않음
- 카카오 프로필 사진은 표시용으로만 사용

## 게스트/홍보

- GUEST도 앱의 분위기와 KDK 흐름을 볼 수 있는 공개/체험 구조 검토
- 가입 전 사용자가 클럽 운영 방식과 경기 시스템을 자연스럽게 이해하도록 구성
- 홍보용 화면은 과장된 랜딩보다 실제 앱 경험을 보여주는 방향 선호
- Preview Mode를 게스트 전환 장치로 활용 가능

## 회원 공개 / 매뉴얼

### 아이디어

- 앱이 어느 정도 안정화되면 테연 클럽 회원들에게 공개하기 전에 사용 매뉴얼을 만든다.
- 회원들이 처음 접속했을 때 로그인, 메뉴 이동, KDK 대진 확인, 라이브 경기 확인, 랭킹 확인 방법을 쉽게 이해할 수 있어야 한다.
- 운영진용 매뉴얼과 일반 회원/게스트용 매뉴얼을 분리하는 것을 검토한다.
- 매뉴얼 제작 방식으로 Figma를 검토한다.
- Figma로 화면 흐름, 클릭 위치, 권한별 사용 방법을 시각적으로 정리하면 회원들에게 공유하기 좋다.
- 최종적으로 PDF, 이미지 카드, 인스타/카카오톡 공유용 요약본으로도 변환할 수 있게 한다.

### 목적

- 회원 공개 시 혼란 줄이기
- 운영진 교육 시간 줄이기
- 앱의 완성도와 신뢰감 강화
- 나중에 다른 클럽 확장/유료화 시 온보딩 자료로 재사용

## 스페셜 매치 / 엑셀 연동 / 결과표

### 아이디어 1: 스페셜 매치 엑셀 대진표 자동 불러오기

- 현재 스페셜 매치 커스텀 모드에서는 대진을 수동으로 입력하고 있다.
- 실제 운영에서는 경기이사가 엑셀로 대진표를 전달하는 경우가 많다.
- 매번 앱에서 수동으로 대진을 입력하는 과정이 번거롭다.
- 향후 엑셀 파일을 업로드하면 앱이 자동으로 대진표를 읽고 스페셜 매치 경기 목록을 생성하도록 하고 싶다.
- 엑셀 구조 예시:
  - 순서
  - 코드/조합 번호
  - 좌측 팀 선수 2명
  - 우측 팀 선수 2명
  - 경기 시간
  - 게스트 표기 예: 이름(G)
- 앱은 엑셀에서 경기 순서, 선수명, 게스트 여부, 경기 시간, 팀 구성을 파싱한다.
- 파싱 후 사용자가 확인/수정할 수 있는 미리보기 화면을 제공한다.
- 최종 저장은 운영진/CEO만 가능하게 한다.
- 목적:
  - 스페셜 매치 운영 준비 시간 단축
  - 엑셀 기반 운영 방식과 앱을 자연스럽게 연결
  - 수동 입력 실수 감소
  - 경기이사가 기존에 쓰던 업무 방식 유지

### 아이디어 2: 스페셜 매치 개인별 상세 결과표

- 현재 카카오톡 복사하기 기능은 최종 순위 요약 텍스트를 제공한다.
- 예시:
  - `[260428_SPECIAL_09] 스페셜 매치 최종 결과`
  - `1위: 정상윤 (4승 0패, 득실 9)`
  - `2위: 김영우 (3승 1패, 득실 12)`
- 이 요약 기능은 유지한다.
- 추가로 개인별 상세 결과표를 앱에서 볼 수 있게 하고 싶다.
- 상세 결과표에는 다음 정보가 포함된다.
  - 순위
  - 성명
  - 승
  - 패
  - 게임 득점: 1세트, 2세트, 3세트, 4세트, 합계
  - 게임 실점: 1세트, 2세트, 3세트, 4세트, 합계
  - 득실
- 두 번째 참고 이미지처럼 개인별 경기 득점/실점과 합계, 득실, 순위를 표 형태로 보여준다.
- 동률일 때 정렬 기준도 명확히 정의해야 한다.
  - 1순위: 승수
  - 2순위: 득실
  - 3순위: 총 득점
  - 4순위: 필요 시 이름 또는 기존 앱 기준
- 이 상세 결과표도 카카오톡 공유용 텍스트 또는 이미지/PDF로 내보내는 것을 장기 검토한다.
- 목적:
  - 경기 후 개인별 성과를 더 투명하게 확인
  - 단순 순위뿐 아니라 득점/실점 흐름까지 확인
  - 회원들이 결과를 납득하기 쉽게 제공
  - 경기 후 공유 자료의 완성도 향상

### 우선순위

- 중상
- 스페셜 매치 운영 자동화와 결과 공유 품질 개선에 직접 연결됨

## TV/모니터용 Live Board

- TV/모니터용 Live Board 화면을 별도 페이지로 만든다.
- 평소 모바일 앱은 그대로 유지하고, 월 2회 4코트 운영 시 이동식 TV에 `/kdk/display` 또는 `/live-board` 화면을 띄운다.
- 이 화면은 읽기 전용이며, 운영진이 모바일에서 경기 투입/점수 입력을 하면 Realtime으로 자동 반영된다.
- TV 화면은 16:9 가로형 레이아웃으로 구성한다.
- 코트별 현재 경기, 다음 경기, 진행 상태, 세션명, 시간, TEYEON 브랜딩을 크게 보여준다.
- 로그인 없이 세션 링크로 볼 수 있는 공개 보기 방식도 검토한다.

## 장기 확장

- `club_id` 기반 멀티 클럽 확장 고려
- 장기적으로 `club_memberships(user_id, club_id, role, member_id)` 구조 검토
- 다른 클럽이 자체 멤버, 권한, KDK 세션을 운영할 수 있는 구조 검토
- 유료화 가능성: 클럽별 운영 도구, 대회 관리, 기록 보관, 프리미엄 디자인
